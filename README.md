# encrypt-leyu-archive

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涉及加密、信息整理或网页资料保存等用途，不针对任何特定域名或具体网站。

## 项目简介

encrypt-leyu-archive 是一个静态 HTML 页面集合仓库。所有页面均为独立文件，无需后端服务支持，可直接在浏览器中打开使用。仓库的主要目的是对某些网页内容进行归档、备份或呈现，方便后续查阅或分享。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 入口页面（可选）
├── pages/             # 归档页面存放目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/            # 页面依赖资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：存放所有归档的 HTML 页面，文件名按归档顺序或内容命名。
- `assets/`：存放页面所需的公共资源文件，如图片、样式表和脚本。

## 页面归档说明

- 每个 HTML 页面均为自包含或引用本仓库内资源的独立文件。
- 页面内容可能包含文字、表格、链接、加密信息等，具体用途由归档者决定。
- 归档的页面通常来自网络公开信息或作者自行整理的内容，不保证时效性、准确性或完整性。
- 任何页面均不代表本仓库作者的立场或观点。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问（需自行配置）。
